# ml-nmr-tutorial
This repo contains material for ML tensorial NMR for the CCP-NC masterclass

You can run the tutorial in your browser using [molab](https://molab.marimo.io). Click the button below to open the tutorial in molab.
[![Open in molab](https://marimo.io/molab-shield.svg)](https://molab.marimo.io/github/cbenmahm/ml-nmr-tutorial/blob/main/tutorial.py)

If you want to run the tutorial locally, you can run it in a sandboxed environment using uvx and marimo:


```bash
uvx marimo edit tutorial.py --sandbox
```
